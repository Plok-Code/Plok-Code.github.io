IDENTITÉ VISUELLE 2.0 — "MODERN INDUSTRIAL PRO"

1. LE DIAGNOSTIC (POURQUOI C'ÉTAIT "MOCHE")
- Trop de "Bebas Neue" partout : Utiliser une police "Display" (titre) pour des petits éléments (menu, boutons) rend la lecture difficile et donne un aspect "amateur" ou "brouillon".
- Manque de hiérarchie : Si tout crie (MAJUSCULES PARTOUT), rien n'est important.
- Boutons "faibles" : Transformer des boutons d'action (CV, LinkedIn) en simples liens texte réduit leur fonction d'appel à l'action.

2. LA NOUVELLE PHILOSOPHIE : "CLARTÉ & IMPACT"
On garde l'âme "Industrial Tokyo" (couleurs sombres, néons) mais on structure la typographie de manière professionnelle.

3. LE SYSTÈME TYPOGRAPHIQUE (DUO GAGNANT)

3.1 TITRES (DISPLAY) -> BEBAS NEUE
- Rôle : Capter l'attention, donner l'ambiance.
- Usage STRICT : Uniquement H1, H2, et très gros chiffres.
- Interdiction : Ne JAMAIS utiliser pour des textes longs ou petits (< 24px).

3.2 UI & LECTURE (FONCTIONNEL) -> INTER (ou MONTSERRAT, plus géométrique)
- Rôle : Lisibilité, Navigation, Crédibilité professionnelle.
- Usage : 
  - Menu de navigation (Clarté).
  - Boutons (Call to Action lisible).
  - Paragraphes (Confort de lecture).
  - Métadonnées.
- Avantage : "Inter" est la référence du design d'interface moderne. Elle est neutre, précise et "Pro".

4. LE HEADER (STRUCTURE PRO)
Au lieu de tout mélanger, on crée des zones distinctes mais harmonieuses :
- Logo (Nom) : Inter Bold (Affirmé).
- Menu : Inter Medium (Lisible, aéré).
- Actions (LinkedIn/CV) : VRAIS BOUTONS.
  - Uniformité de hauteur : Liens menu et Boutons auront la même *hauteur visuelle* (alignement optique).
  - Différenciation subtile : Les boutons auront une bordure fine ou un fond léger pour inviter au clic, mais avec la même *police* que le menu pour ne pas jurer.

5. PALETTE (RAFFINÉE)
- Fond : #0E0F14 (Toujours ce noir profond "Tokyo").
- Texte : #E2E4E9 (Blanc cassé premium, moins dur que #FFF).
- Accent : #38BDF8 (Sky Blue) pour les liens/actions -> Plus moderne que le Cyan pur.

6. PLAN D'ACTION IMMÉDIAT
- [HTML] Réintégrer "Inter" (la meilleure font pour l'UI).
- [CSS] Réassigner les rôles : 
  - Menu = Inter.
  - Titres = Bebas.
  - Boutons = Inter (Style "Button", pas "Lien").
- [DESIGN] Calibrer les espacements pour que le Header respire.
